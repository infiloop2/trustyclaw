# Google integration guide image sources

These shared Gmail and Google Calendar release assets are local so the admin UI
never loads provider-controlled content. They are unmodified screenshots from
Google's current OAuth setup codelab, retrieved 2026-07-13:

- `google-auth-app-information.png`:
  `https://codelabs.developers.google.com/sign-in-with-google-android/images/GCP_app_info.png`
- `google-auth-data-access.png`:
  `https://codelabs.developers.google.com/static/ge-gws-agents/img/874b1dda14e8f379.png`
- `google-auth-web-client.png`:
  `https://codelabs.developers.google.com/sign-in-with-google-android/images/finish_web_client.png`

The corresponding setup steps link to Google's maintained Workspace OAuth
documentation for the authoritative instructions.
